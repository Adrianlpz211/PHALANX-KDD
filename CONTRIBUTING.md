# Contributing to Agentic KDD

Thank you for your interest in contributing.

## Ways to contribute

- **Report bugs** — open an issue with steps to reproduce
- **Suggest improvements** — open an issue with your idea
- **Improve agent instructions** — PRs to `.agentic/agentes/` files
- **Add stack examples** — show how KDD works with your stack
- **Improve documentation** — README, methodology docs, examples

## Pull Request process

1. Fork the repo
2. Create a branch: `git checkout -b feature/your-improvement`
3. Make your changes
4. Open a PR with a clear description of what changed and why

## Issues

Use the issue templates for bugs and feature requests.
